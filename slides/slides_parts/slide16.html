DEVDAY 2025

AGENT — Key Features & Core Concepts

Fonctionnalités clés des Workflows

  • Type Safety : Typage fort pour valider le flux des messages entre composants
  • Flexible Control Flow : Architecture à base de graphes (executors + edges)
  • External Integration : Patterns request/response pour APIs externes et human-in-the-loop
  • Checkpointing : Sauvegarder l'état pour reprendre les processus long-running
  • Multi-Agent Orchestration : Patterns built-in (sequential, concurrent, hand-off, magentic)

Core Concepts

Executors

Unités de traitement (agents ou logique custom) qui reçoivent des messages, exécutent des tâches, et produisent des outputs.

Edges

Connexions entre executors définissant le flux des messages. Peuvent inclure des conditions pour le routage.

Workflows

Graphes dirigés composés d'executors et edges, définissant le processus global avec routage conditionnel.
