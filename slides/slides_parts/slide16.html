AGENT — Les Patterns d'Orchestration

Handoff

Relais entre agents spécialisés

Triage → Expert → Escalation

Concurrent

Plusieurs agents en parallèle

Research + Analysis + Summary

Conditional Routing

Routage dynamique selon résultats

if confidence > 0.8 → Auto
else → Human

Exemple : Support Client Automatisé

User Question
↓
TriageAgent (catégorise: technique/facturation/autre)
⤴ technique → TechAgent (RAG sur docs)
⤴ facturation → BillingAgent (accès DB)
⤴ confidence < 0.7 → EscalationAgent (handoff humain)
