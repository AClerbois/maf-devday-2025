CHECK — Piège #3 : Ne Pas Gérer le Contexte ⚠️

❌ Sans Memory

Tout dans AgentThread
→ Explosion tokens
→ Perte infos anciennes
→ Coûts incontrôlables

❌ Sans AgentThread

L'agent oublie tout
→ Pas de cohérence
→ Répétitions
→ Expérience frustrante

✅ Les deux ensemble

  • AgentThread pour cohérence conversationnelle immédiate (10 derniers messages)
  • Memory pour rappel sémantique des informations anciennes
  • Stratégie : Summariser régulièrement le thread et archiver dans Memory
Message 1-10: AgentThread → Summarize → Memory
Message 11-20: AgentThread → Recall from Memory si pertinent
