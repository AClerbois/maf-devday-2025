PLAN — Configuration d'un Agent

AIAgent agent = new AzureOpenAIClient( new Uri("https://devday-2025-maf.openai.azure.com/"), new AzureCliCredential() ) .GetChatClient("mistral-medium-2505") .CreateAIAgent( instructions: "Tu es un spécialiste de film popculture.", name: "Joker", description: "Un agent qui adore parler de films et de séries." );

Points clés

  • Instructions : Le "cerveau" de l'agent, définit sa logique de raisonnement
  • Name : Identifier l'agent dans les workflows multi-agents
  • Description : Aide d'autres agents à comprendre quand l'utiliser
