PLAN — Configuration d'un Agent

var agent = new ChatCompletionAgent(chatClient) { Instructions = @"Tu es un assistant spécialisé en recherche technique. Ton rôle : - Analyser les questions techniques - Décomposer en sous-problèmes - Chercher dans la documentation - Synthétiser clairement Règles : - Cite toujours tes sources - Admets quand tu ne sais pas - Reste concis", Name = "ResearchAgent", Description = "Agent de recherche technique" };

Points clés

  • Instructions : Le "cerveau" de l'agent, définit sa logique de raisonnement
  • Name : Identifier l'agent dans les workflows multi-agents
  • Description : Aide d'autres agents à comprendre quand l'utiliser
