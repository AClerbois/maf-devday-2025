CHECK — Le système Memory en détail

Le problème sans Memory

Sans Memory, tout va dans AgentThread → explosion des tokens → perte de contexte ancien → coûts incontrôlables

Comment fonctionne Memory ?

  1. Vectorisation : Les faits/conversations importants sont convertis en embeddings
  2. Stockage : Base vectorielle (Azure AI Search, Qdrant, etc.)
  3. Rappel sémantique : Seules les infos pertinentes sont injectées dans le contexte
